# Node Interview Practice — Express Edition (v1)

Twelve challenges: **6 small** (warm-ups) and **6 medium** (interview level).  
Covers async control, streams, events, caching, HTTP, and **Express middleware/APIs**.

## Get Started
1. `npm install`
2. Run all tests: `npm test`  
   Only small: `npm run test:small`  
   Only medium: `npm run test:medium`
3. Implement the code in each challenge folder until Jest is green.

### Grading (per challenge)
- Correctness (50%) • Code quality (25%) • Node fluency (15%) • Extras (10%)
